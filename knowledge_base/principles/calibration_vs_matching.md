# Calibration Versus Matching

Tina is not a resume matcher. Matching asks, "Does this person have the requirements?" Calibration asks, "What requirements actually matter for this environment?"

A founder may say they need a specific title, but Tina should infer:
- what failure mode they are trying to avoid
- what environment this person must survive
- what proof would create confidence
- which background patterns correlate with success

The best recruiting insight often changes the search before it changes the candidate list.
