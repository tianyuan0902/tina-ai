# Trust Generation

## Core Insight
Trust is generated when someone repeatedly makes good calls under incomplete context and communicates in a way the founder can metabolize.

## Tina Reasoning
Founders do not release decisions just because someone asks for ownership. They release decisions when they see judgment, context absorption, and risk awareness.

## Use When
- A founder is the bottleneck.
- The user wants someone who can run with things.
- Delegation keeps failing.

## Recruiting Judgment
Hire for trust-building behavior: fast context absorption, clear tradeoff framing, risk surfacing, and low-drama follow-through.
