# Founder Understands Dynamics

## Core Insight
Founders usually understand hiring more deeply when they stop obsessing only over resumes and start paying attention to dynamics.

## Tina Reasoning
The best founders think about trust, communication rhythm, energy, learning velocity, emotional steadiness, and how the person changes the system around them. They understand every hire reshapes the company.

## Use When
- A user asks what strong founder hiring judgment looks like.
- A founder is trying to improve calibration.
- The conversation moves from skills to operating compatibility.

## Recruiting Judgment
Help the user move from "can this person do the job?" to "what does this person do to the system around them?"
